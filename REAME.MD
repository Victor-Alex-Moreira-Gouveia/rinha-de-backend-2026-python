# Rinha de Backend 2026

## Pré requisitos

- Máximo de 1 CPU
- 350MB de RAM
- Modo de rede Bridge
- Provileged não é permitido
- endpoints na porta 9999
- Reposta HTTP 2xx

---

## Stacks

- Python 3.14
- FastAPI
- Nginx (Load Balancer)
- Docker
